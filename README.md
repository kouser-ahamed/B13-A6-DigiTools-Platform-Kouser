## 💡 Project Overview

DigiTools is a **modern, responsive web application** that showcases and sells premium digital tools and services on a single platform.  

- 💻 **Smooth Browsing Experience:** Navigate and explore products easily with a responsive and clean layout.  
- 🛒 **Interactive Cart System:** Add, remove, and manage products seamlessly with real-time updates.  
- 🎨 **Professional UI Design:** Modern, aesthetically pleasing interface suitable for SaaS or digital marketplaces.  
- ⚡ **Real-world Frontend Practices:** Built using React and Tailwind CSS with component-based architecture for scalability.

---

## 🛠️ Technologies Used

- ⚛️ **React.js** – Build reusable and modular UI components.  
- 🟨 **JavaScript (ES6+)** – Modern syntax for efficient and clean code.  
- 🎨 **Tailwind CSS** – Rapidly create responsive and stylish layouts.  
- 🔄 **React Hooks (`useState`, `useEffect`)** – Manage state and side-effects efficiently.  
- 🖋️ **React Icons** – Beautiful, scalable icons for UI enhancements.  
- 🔔 **React Toastify** – Smooth toast notifications for user feedback.  
- 🚀 **Vite** – Fast development and build environment.  
- 📄 **JSON Data Fetching** – Load product data asynchronously for dynamic updates.

---

## 📦 Core Features

1. 🛍️ **Dynamic Product Listing**  
   - 📂 Load products dynamically from a JSON file asynchronously.  
   - 📝 Each card displays **name, description, price, features, and tags** (new, popular, best seller).  
   - 🔧 **Reusable Card Component:** Modular design for maintainability and scalability.

2. 🛒 **Cart Management System**  
   - ➕ Add or remove products seamlessly.  
   - ❌ Prevent duplicate items with proper validation.  
   - 💰 Automatic total price calculation displayed in real-time.  
   - ✅ Checkout clears the cart and provides success feedback.

3. 🎨 **Modern UI & Responsive Design**  
   - 📱 Fully responsive for mobile, tablet, and desktop devices.  
   - 🧭 Custom navigation bar with mobile menu support.  
   - ✨ Smooth hover effects, transitions, and animations using Tailwind CSS.  
   - 🏗️ Professional sections: Hero Banner, Pricing Plans, Workflow, and Getting Started Steps.

## 🔧 Local Setup Guide

1. 📥 **Clone the repository**  
```bash
git clone https://github.com/kouser-ahamed/B13-A6-DigiTools-Platform-Kouser.git
````

2. 📂 **Navigate to the project folder**

```bash
cd digitools
```

3. ⚙️ **Install dependencies**

```bash
npm install
```

4. 🚀 **Start the development server**

```bash
npm run dev
```

5. 🌐 **Open in your browser**
   Go to [http://localhost:5173](http://localhost:5173) to view the application.
